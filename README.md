# EncryptedFourInARow Online
This is an accademic project and wants to be a simple and fun example in how to use cryptographic tools to develop a secure protocol.
You can find examples of using:
- **RSA**
- **ECDH Key Exchange**
- **GCM** mode
- Secure Coding
- Use of **Certificates**
- **AES**
